== Description ==
<h3>Rio Portfolio</h3>
It is a Portfolio management plugin that allows you to manage Image, Link and Video type Portfolio items with jquery filtration option. You can use just '[rio-portfolio]' shortcode to display all Portfolios. Using this plugin you can open Portfolios in a pop up


= Plugin Features =



1. Custom post type for Portfolio items
1. Video, Image, Link and Default post format support
1. Youtube, Vimeo and Dailymotion video support
1. Using PrettyPhoto gallery for Images and Videos
1. Easy categorization and jQuery sorting feature
1. Pagination with jquery filteration
1. Powerful backend for managing Portfolio page
1. Post order option for portfolio items
1. Manage thumbnail size from back end
1. Help option for managing Portfolios

== Installation ==

This section describes how to install the plugin and get it working

1. Upload `rio-portfolio` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[rio-portfolio]` in your any page or post
