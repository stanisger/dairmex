This plugin adds a custom favicon a badge showing the number of products that are in the shopping cart

== Description ==
This plugin displays custom favicon with badge for number of woocommerce cart products

Configuration parameters: 

1. **Favicon Image** (upload or url).
2. **bgColor**:		*default: #d00*, Options: Badge background color.
3. **textColor**:	*default: #fff*, Options: Badge text color.
4. **fontFamily**:	*default: sans-serif*, Options: Text font family (Arial, Verdana, Times New Roman, serif, sans-serif,...).
5. **fontStyle**:	*default: bold*, Options: Font style (normal, italic, oblique, bold, bolder, lighter, 100, 200, 300, 400, 500, 600, 700, 800, 900).
6. **type**:		*default: circle*, Options: Badge shape (circle, rectangle).
7. **position**:	*default: down*, Options: Badge position (up, down).
8. **animation**:	*default: slide*, Options: Badge animation type (slide, fade, pop, popFade, none).

== Installation ==
1. Upload "woocommerce-cart-badge" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
